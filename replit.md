# Notemaker

A Flask-based academic study tool that uses AI to generate structured subtopics and detailed notes for any given topic.

## Features
- **Topic Generation**: Get 8-12 structured subtopics for any academic subject.
- **Detailed Notes**: Deep dive into any subtopic with AI-generated markdown notes.
- **Daily Limits**: Built-in usage tracking to manage API costs.

## Tech Stack
- **Backend**: Python, Flask
- **AI**: Groq (Llama 3 8B)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Formatting**: Marked.js for markdown rendering

## Setup
1. Set `GROQ_API_KEY` in your environment.
2. Run `python main.py`.
3. Open `http://localhost:5000`.
